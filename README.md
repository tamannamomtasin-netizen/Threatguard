# Threatguard
A Lightweight Intrusion Detection &amp;   Secure Authentication System
Future improvement to ThreatGuard would include incorporating machine learning intrusion detection, multi-factor authentication, live monitoring dashboard, database, email notifications, role-based access control, improved password hashing, network traffic analysis, account lockout features, and cloud-based monitoring. With the inclusion of these features, the system will become more secure, smart, and applicable in actual cybersecurity practice.
